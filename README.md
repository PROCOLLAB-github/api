# Procollab Backend

Backend API для продукта Procollab.

## Стек

- Python
- Django
- Django REST Framework
- Channels
- Celery
- PostgreSQL
- Redis

## Базовые команды

```bash
poetry install
poetry run python manage.py migrate
poetry run python manage.py runserver
poetry run python manage.py test
```
